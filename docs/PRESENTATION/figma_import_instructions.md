# Cómo importar la presentación a Figma

1. Descarga `aurora_presentation.pptx` desde `docs/PRESENTATION/`.
2. Abre Figma > File > Import > selecciona el PPTX.
3. Ajusta tipografías y colores al tema Aurora (Dark: #0b1220, Accent: #6EE7B7).
4. Para interactividad añade links entre slides y componentes.
